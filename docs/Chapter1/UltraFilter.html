超フィルターとは

既にフィルター全体が包含に関して順序構造を持つことを見た。このとき与えられた2つのフィルターについて、下限は常に存在するが、上限は必ずしも定義されない。この事実から、大きなフィルターには特別な意味があると推察できる。

定義 包含順序に関して極大なフィルターを超フィルター（ultra filter）と呼ぶ。

定理 任意のフィルターF⊂2X{ \mathscr{F}\subset 2^{X} }F⊂2Xについて、F{ \mathscr{F} }Fを含む超フィルターが存在する。

（証明）X{ X }Xのフィルター全体をΦ(X){ \Phi( X ) }Φ(X)とする。Ψ:={G∈Φ(X):F⊂G}{ \Psi:=\lbrace \mathscr{G}\in\Phi( X ) : \mathscr{F}\subset\mathscr{G} \rbrace }Ψ:={G∈Φ(X):F⊂G}の全順序部分集合τ{ \tau }τを取れば、⋃A∈τA{ \bigcup_{\mathscr{A}\in\tau}\mathscr{A} }⋃A∈τ​Aはフィルターであり、更にτ{ \tau }τのΨ{ \Psi }Ψにおける上界を与えている。故にΨ{ \Psi }Ψは帰納的であり、Zornの補題より極大元U{ \mathscr{U} }Uを持つ。これはF{ \mathscr{F} }Fを含む超フィルターである。□{ \square }□
